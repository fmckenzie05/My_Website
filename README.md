## 🚀 Local Development

### Prerequisites
- Python 3.x (for local server)
- Modern web browser
- Git

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/fmckenzie05/My_Website.git
   cd My_Website
   ```

2. **Start local server**
   ```bash
   python -m http.server 8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```
